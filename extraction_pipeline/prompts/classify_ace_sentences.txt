DECISION PROCEDURE — apply these steps in order to select the category:

Step 1: Is the sentence about the visualization artifact itself (chart title, source, publisher, task)?
  → Yes → Meta / Paratext (11)

Step 2: Is the sentence describing something visible IN the chart?
  → Yes → continue to Steps 2a–2c
  → No  → go to Step 3

  Step 2a: Does it describe chart structure, layout, text labels, what a symbol/color means, or the presence/absence of visual elements — WITHOUT extracting a numerical data value or comparing data points?
    → Yes → Visual Observation: Chart Structure & Text (1)
  Step 2b: Does it extract a specific numerical value, quantity, or magnitude for ONE data point or entity?
    → Yes → Visual Observation: Data Point Extraction (2)
  Step 2c: Does it compare two or more data points, trace a trend over time, describe a spatial distribution, characterize the overall shape/spread of data, or identify a correlation — all grounded in what the chart visually shows?
    → Yes → Visual Observation: Cross-point Pattern Recognition (3)

Step 3: Is it a personal anecdote, memory, or self-reference?
  → Yes → Prior Knowledge: Personal / Episodic (5)

Step 4: Is it primarily a normative stance, emotional reaction, or value judgment?
  → Yes → continue to Steps 4a–4b
  → No  → go to Step 5

  Step 4a: Does the sentence contain a deontic operator (should, must, need to, ought to), a moral judgment (unfair, wrong, "a problem"), or a recommendation for action?
    → Yes → Evaluative: Prescriptive (6)
  Step 4b: Does the sentence express an emotional reaction, general approval/disapproval, or evaluative stance WITHOUT prescribing what should change?
    → Yes → Evaluative: Reactive (7)

Step 5: Is it an explicit question, expression of uncertainty, or desire for information?
  → Yes → Curiosity (10)

Step 6: Does it propose WHY something happened — linking an observation to a cause?
  → Yes → Inference: Explanatory (8)

Step 7: Does it state a fact about the world not shown in the chart (without causal linking)?
  → Yes → Prior Knowledge: Background (4)

Step 8: Is it future-oriented, hypothetical, or counterfactual?
  → Yes → Inference: Predictive / Hypothetical (9)

Step 9: None of the above → Uncategorizable (12)

DISAMBIGUATION RULES for commonly confused pairs:

- Visual Observation: Chart Structure & Text vs. Data Point Extraction: If the statement describes structure, layout, text labels, or what a symbol/color means, it is Chart Structure & Text. If it extracts a specific numerical value or quantity, it is Data Point Extraction. Example: "The darker pink bars represent the players of color" → Chart Structure & Text (symbol meaning); "The NBA has 83% of players of color" → Data Point Extraction (specific value).
- Visual Observation: Data Point Extraction vs. Cross-point Pattern Recognition: If the statement is about ONE data point or entity, prefer Data Point Extraction. The moment it holds TWO or more data points in mind to compare, trace a trend, or characterize a pattern, prefer Cross-point Pattern Recognition. Example: "Less than 25% of young men visit in person" → Data Point Extraction (single value); "Women communicate with their parents more than men in all forms" → Cross-point Pattern Recognition (comparison across groups).
- Visual Observation: Cross-point Pattern Recognition vs. Inference: Explanatory: If the sentence only describes WHAT the chart shows (a comparison, trend, or pattern), it is Cross-point Pattern Recognition. If it proposes WHY the pattern exists (invoking a cause), it is Inference: Explanatory — even if the pattern is mentioned in the same sentence.
- Visual Observation: Cross-point Pattern Recognition vs. Inference: Predictive / Hypothetical: If the statement describes what the chart visually depicts (observed data), it is Cross-point Pattern Recognition. If it speculates about the future, projects beyond the data, or proposes hypothetical scenarios, it is Inference: Predictive / Hypothetical — not a visual observation.
- Prior Knowledge: Background vs. Inference: Explanatory: If the sentence states a standalone fact without connecting it to chart data, it is Prior Knowledge: Background. If it links that fact to an observation (even implicitly with "may", "because", "relates to"), it is Inference: Explanatory.
- Evaluative: Prescriptive vs. Evaluative: Reactive: If the sentence contains a deontic operator (should, must, need to, ought to), a moral judgment term (unfair, wrong), identifies a "problem" requiring change, or recommends action, it is Prescriptive. If the sentence expresses an emotional reaction (shock, worry, sadness, hope, disappointment) or general approval/disapproval without prescribing change, it is Reactive. Tie-breaker: if both are present, classify by primary communicative function — advocating for change → Prescriptive; expressing a feeling → Reactive.
- Evaluative vs. Inference: Explanatory: If the sentence primarily expresses a value judgment or emotional reaction, prefer Evaluative (Prescriptive or Reactive). If it primarily proposes a cause, prefer Inference: Explanatory — even if the tone is evaluative.
- Prior Knowledge: Background vs. Prior Knowledge: Personal / Episodic: If the sentence is about the self, one's community, or a personal memory, it is Personal / Episodic. If it states a general fact about the world, it is Background.
